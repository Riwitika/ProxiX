# ProxiX

ProxiX is a secure attendance platform designed to eliminate proxy attendance in educational institutions. The system uses dynamic QR code sessions combined with one-person-one-login authentication. Students must verify their identity through face recognition and fingerprint verification before attendance is recorded, ensuring only the actual student present can mark attendance.

## Features
- Dynamic rotating QR attendance sessions
- One-person-one-login authentication
- Face recognition verification
- Fingerprint authentication
- Real-time attendance tracking
- Teacher dashboard

## Tech Stack
- React
- Supabase
- Node.js
- TailwindCSS
